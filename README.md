# Ways_to_improve_the_quality_of_education

## Описание проекта
Данный репозиторий содержит материалы для лабораторной работы 'Способы повышения качества обучения' связанной с анализом данных и применением алгоритмов машинного обучения. В качестве предметной области используется анализ медицинских данных — датасет `heart.csv`, который классически применяется для задач бинарной классификации (предсказания вероятности сердечных заболеваний на основе клинических показателей).

## Структура репозитория
* **`MLP_ways_to_improve_the_quality_of_education.ipynb`** — основной Jupyter Notebook. В нем реализованы этапы загрузки данных, разведочного анализа (EDA), предобработки и построения прогнозных моделей.
* **`heart.csv`** — набор данных, содержащий информацию о пациентах (такие признаки, как возраст, уровень холестерина, артериальное давление и т.д.).
* **`README.md`** — описание проекта.

## Требования к окружению
Для запуска проекта рекомендуется использовать **Python 3.12** (версия, зарекомендовавшая себя как стабильная для текущих ML-библиотек). 

Необходимые зависимости:
* `pandas`
* `numpy`
* `scikit-learn`
* `matplotlib`
* `seaborn`
* `jupyter`

## Установка и запуск
1.  Склонируйте репозиторий на свой локальный компьютер:
    ```bash
    git clone https://github.com/coolcode17/Ways_to_improve_the_quality_of_education.git
    ```
2.  Перейдите в директорию проекта:
    ```bash
    cd Ways_to_improve_the_quality_of_education
    ```
3.  Создайте виртуальное окружение и установите необходимые библиотеки:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для macOS/Linux
    venv\Scripts\activate     # Для Windows
    pip install pandas numpy scikit-learn matplotlib seaborn jupyter
    ```
4.  Запустите Jupyter Notebook:
    ```bash
    jupyter notebook
    ```
5.  Откройте файл `MLP_ways_to_improve_the_quality_of_education.ipynb` и выполните ячейки по порядку.

## Цели лабораторной работы
* Изучение структуры набора данных `heart.csv`.
* Подготовка данных для обучения: обработка пропусков (если имеются), масштабирование признаков и кодирование категориальных переменных.
* Реализация и сравнение алгоритмов машинного обучения.
* Оценка точности моделей на основе метрик классификации.
