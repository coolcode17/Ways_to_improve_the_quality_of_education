# Ways_to_improve_the_quality_of_education

# Анализ факторов качества образования и путей их оптимизации
# Описание проекта:
Данный проект посвящен разработке и оценке моделей глубокого обучения для классификации состояния здоровья сердца на основе медицинских показателей. В проекте исследуются различные архитектуры нейронных сетей, методы оптимизации и регуляризации для достижения максимальной точности предсказания.

# Используемый датасет:
Проект использует датасет heart.csv.
Целевая переменная (target): Наличие заболевания сердца (1 — болен, 0 — здоров).
Признаки (X): Различные биометрические показатели (возраст, пол, тип боли в груди, давление и т.д.).

Препроцессинг:
Разделение на обучающую и тестовую выборки в соотношении 80/20.
Стандартизация признаков с помощью StandardScaler для улучшения сходимости нейронной сети.

# Главная проблема:
Необходимость определения наиболее значимых рычагов воздействия на систему образования: что эффективнее — прямое финансирование материальной базы или инвестиции в квалификацию кадров и социальную поддержку?

# Ключевой вывод:
# Оптимизация и гиперпараметры
*   **Критическая роль подбора**: Успех модели напрямую зависит от правильной настройки гиперпараметров — таких как количество слоев, плотность нейронов и сила регуляризации.
*   **Баланс обучения**: Точная настройка коэффициента обучения (learning rate) и использование планировщиков (`ReduceLROnPlateau`) позволяют избежать застревания в локальных минимумах.
*   **Контроль сложности**: Тонкий подбор параметров Dropout и L2-регуляризации необходим для поиска «золотой середины» между недообучением и переобучением.
*   **Эффективность архитектуры**: Экспериментальный подбор структуры сети подтверждает, что даже небольшие изменения в конфигурации слоев существенно влияют на итоговую точность предсказания.

## Структура репозитория
* **`MLP_ways_to_improve_the_quality_of_education.ipynb`** — основной Jupyter Notebook. В нем реализованы этапы загрузки данных, анализа, предобработки и построения прогнозных моделей.
* **`heart.csv`** — набор данных, содержащий информацию о пациентах (такие признаки, как возраст, уровень холестерина, артериальное давление и т.д.).
* **`README.md`** — описание проекта.

## Требования к окружению
Для запуска проекта рекомендуется использовать **Python 3.12** (версия, зарекомендовавшая себя как стабильная для текущих ML-библиотек). 

Необходимые зависимости:
* `pandas`
* `numpy`
* `scikit-learn`
* `matplotlib`
* `seaborn`
* `jupyter`

## Установка и запуск
1.  Склонируйте репозиторий на свой локальный компьютер:
    ```bash
    git clone https://github.com/coolcode17/Ways_to_improve_the_quality_of_education.git
    ```
2.  Перейдите в директорию проекта:
    ```bash
    cd Ways_to_improve_the_quality_of_education
    ```
3.  Создайте виртуальное окружение и установите необходимые библиотеки:
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для macOS/Linux
    venv\Scripts\activate     # Для Windows
    pip install pandas numpy scikit-learn matplotlib seaborn jupyter
    ```
4.  Запустите Jupyter Notebook:
    ```bash
    jupyter notebook
    ```
5.  Откройте файл `MLP_ways_to_improve_the_quality_of_education.ipynb` и выполните ячейки по порядку.

## Цели лабораторной работы
* Изучение структуры набора данных `heart.csv`.
* Подготовка данных для обучения: обработка пропусков (если имеются), масштабирование признаков и кодирование категориальных переменных.
* Реализация и сравнение алгоритмов машинного обучения.
* Оценка точности моделей на основе метрик классификации.
